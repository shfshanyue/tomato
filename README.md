# tomato

> 由 Vue 编写的 Web App，使用 cordova 打包。

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for android
cordova build android
```
